# 💱 Currency Exchange

A simple Currency Exchange web application that converts one currency into another using a live exchange rate API.

## 🚀 Features

- Live Currency Exchange Rates
- 160+ Currencies
- Country Flags
- Responsive Design
- Easy to Use

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Currency API
